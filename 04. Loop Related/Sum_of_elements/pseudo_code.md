1. DEFINE FUNCTION get_sum(nums):

   - SET sum to 0
   - FOR each num in nums DO:
     - ADD num to sum
   - RETURN sum

2. SET nums to the list [13, 89, 65, 42, 12, 11, 56]

3. CALL get_sum(nums) and STORE the result in total

4. DISPLAY "The total of each elements: " followed by total
